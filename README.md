## Démarrage du jeu

Télécharger le dossier Projet-D-mineur et le mettre sur le bureau.
Dans un terminal exécutez les commandes suivantes pour lancer le jeu:

cd Desktop

cd Projet-D-mineur

cd JeuDemineur

python Play.py

Commande pour jouer:

r = révéler une case

f = placer ou retirer un drapeau

q = quitter le jeu
